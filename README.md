# go-chatgpt

# ChatGPT in Go TUI.

---

## How to use

### You need to set two env variables

| key          | value         | remark             |
|--------------|---------------|--------------------|
| ACCESS_TOKEN | eyJxxx        | Your access token. |
| API_KEY      | skxxx         | Your api key.      |
| SERVER_URL   | http(s)://xxx | Your server url.   |

---

### Shortcuts

| key | remark                                                          |
|-----|-----------------------------------------------------------------|
| F2  | Select on conversation title, popup a modal to input new title. |
